# **App Name**: SaaS Lifecycle Optimizer

## Core Features:

- Event Capture via SDK: Capture real-time product events via lightweight SDK embedded in SaaS products.
- Lifecycle State Engine: Automatically classify users and accounts into dynamic lifecycle states based on product events and behavior.
- Churn Risk Prediction: AI tool that calculates a churn risk score for each user and account, identifying those at high risk of churn based on behavioral patterns. LLM uses reasoning to incorporate usage statistics.
- Automated Email Flows: Trigger revenue-aware email marketing flows based on lifecycle states, risk scores, and expansion opportunities.
- Revenue Attribution Dashboard: Track MRR per user, ARR per account, and revenue per lifecycle state to measure the impact of email flows.
- Client Admin Dashboard: Built in Next.js, the dashboard will allow users to quickly see key metrics relating to Activation, Retention, and Expansion
- Account-Level Targeting: Allows emails to target based on roles defined at the account level

## Style Guidelines:

- Primary color: Deep sky blue (#42A5F5) for a SaaS-focused, trustworthy, and engaging feel.
- Background color: Very light blue (#E3F2FD) providing a clean, modern backdrop.
- Accent color: Purple (#9C27B0) to highlight key actions and metrics, differentiating them from the primary color.
- Body and headline font: 'Inter' sans-serif, to maintain a modern, machined, objective, neutral feel.
- Use minimalist icons to represent lifecycle stages and revenue metrics.
- Emphasize clear, data-driven dashboards with intuitive navigation. Design with a 'revenue-first clarity', and without marketing clutter.
- Subtle transitions for lifecycle changes and data updates to enhance user experience.
